# SQL Sales Data Analysis
## Overview

This project analyzes retail sales data using SQL (MySQL) to extract meaningful business insights.

## Tools Used
- MySQL
- MySQL Workbench
## Analysis Performed
- Total Sales Calculation
- Sales by Category
- Sales by Region
- Top Customers Analysis
- Year-wise Sales Trend
## Key Insights
- Technology is the top-performing category
- West region generates the highest revenue
- Sales increased significantly after 2015
- A few customers contribute a large portion of total revenue

- Dataset (CSV)
- SQL Queries (queries.sql)
